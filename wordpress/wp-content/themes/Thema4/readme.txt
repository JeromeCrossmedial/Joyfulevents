== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How to setup the front page like the demo site? =

The demo site URL: http://demo.ilovewp.com/photoframe/

When you first activate Photoframe, your homepage will display posts in a traditional blog format. If you would like to add the additional featured sections from the demo, you can do that by going to Appearance > Customize > Theme Settings.

= Footer Area =

The footer area contains 3 widgetized columns of equal width.

== Quick Specs (all measurements in pixels) ==

	1. The main column width is 660.
	2. In full-width layout the content width is 1100.
	3. Featured Images for posts should be at least 320 wide (for archive pages).
	4. Featured Images for posts and pages should be at least 1600 if hero images are enabled in Appearance > Customize > Theme Settings.
	5. The footer widget columns are of equal width: 320.
